# 🎮 Conversor de Moedas Virtuais

Este projeto é uma aplicação web simples e intuitiva desenvolvida para converter valores em **Reais (R$)** para moedas de jogos populares como **Roblox (Robux)** e **Fortnite (V-Bucks)**. O projeto foi criado com base na lógica de manipulação de dados e feedback dinâmico.

---

## 🚀 Como funciona?

O conversor utiliza cálculos matemáticos baseados em fatores de conversão simulados:

* **Roblox:** Cada R$ 1,00 equivale a **16 Robux**.
* **Fortnite:** Cada R$ 1,00 equivale a **25 V-Bucks**.

O sistema valida a entrada do usuário para garantir que apenas números positivos sejam calculados. Caso o valor seja inválido, o sistema emite um alerta visual na cor vermelha. Se os dados estiverem corretos, ele exibe o resultado estilizado com as cores temáticas de cada jogo.

---

## 🛠️ Tecnologias Utilizadas

Para este projeto, foram utilizadas as tecnologias fundamentais da web:

1.  **HTML5:** Estrutura dos formulários, seletores e áreas de resultado.
2.  **CSS3:** Estilização com design "Dark Mode", layout responsivo e transições suaves.
3.  **JavaScript:** Lógica de captura de eventos, cálculos de conversão e manipulação do DOM para exibição dinâmica.

---

## 📂 Como Acessar o Projeto

Você não precisa instalar nada complexo para rodar este projeto, apenas um navegador de internet.

1.  **Baixe o código:** Salve o código fornecido em um arquivo chamado `index.html`.
2.  **Abra o arquivo:**
    * Localize o arquivo no seu computador.
    * Clique com o botão direito e selecione **"Abrir com"**.
    * Escolha o seu navegador de preferência (Google Chrome, Firefox, Edge, etc.).
3.  **Use o sistema:** Digite o valor no campo de entrada, escolha o jogo no seletor e clique no botão para ver a mágica acontecer!
